# Project: Aspirin Molecule Analysis

This project analyzes the chemical properties of Aspirin using RDKit.

## Files

- `analyze_molecule.py`: A Python script to calculate key molecular descriptors.
- `explore_aspirin.ipynb`: A Jupyter Notebook for interactive visualization.

## Dependencies

- Python 3.8+
- RDKit

## Usage

Run the main script from the terminal:
```bash
python analyze_molecule.py
```
